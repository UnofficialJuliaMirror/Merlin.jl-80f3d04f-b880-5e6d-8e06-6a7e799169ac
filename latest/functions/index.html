# Merlin.maxpooling — Function.

maxpooling(window, [stride, padding])

Arguments

  • windims::NTuple{N,Int}: window size
  • stride::NTuple{N,Int}: stride size. Default: (1,1,...)
  • paddims::NTuple{N,Int}: padding size. Default: (0,0,...)
