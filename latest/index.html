Merlin.jl

Merlin is a flexible deep learning framework written in Julia. It aims to provide a fast, flexible and compact deep learning library for machine learning.

Requirements

  • Julia 0.6

  • g++ (for OSX or Linux)

Installation

First, install Julia. Currently, version 0.6 is supported.

Then, clone and build the package.

julia> Pkg.clone("https://github.com/hshindo/Merlin.jl.git")
julia> Pkg.build("Merlin")
