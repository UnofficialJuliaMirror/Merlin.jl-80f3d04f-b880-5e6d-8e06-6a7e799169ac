Merlin.jl

Merlin.jl is a flexible neural network library in Julia language.

Install

julia> Pkg.clone("https://github.com/hshindo/Merlin.jl.git")

Requirements

  • gcc
  • cuDNN v4 (if you use CUDA GPU)
