Merlin.jl

Merlin.jl is a neural network library in Julia.

It aims to provide a fast, flexible and concise neural network library for machine learning.

Merlin.jl uses our customized arrayfire library for array computation on CPU / GPU / OpenCL backends through ArrayFire.jl.

★ Install

julia> Pkg.clone("https://github.com/hshindo/Merlin.jl.git")

★ Optional Requirements

  • cuDNN v4 (for CUDA GPU)
