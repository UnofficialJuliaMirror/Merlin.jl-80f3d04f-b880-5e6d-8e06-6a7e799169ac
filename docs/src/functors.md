# Functors

`Functor` is an abstract type of function object.
Every `Functor` type implements forward and backward computation.
The following are concrete types of `Functor`s.

    {docs}
    Concat
    CrossEntropy
    Linear
    LogSoftmax
    Lookup
    Max
    ReLU
    Reshape
    Sigmoid
    Softmax
    Tanh
    Window2D
