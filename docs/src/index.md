# Merlin.jl

[Merlin.jl](https://github.com/hshindo/Merlin.jl) is a neural network library in [Julia](http://julialang.org/).

## Install
```julia
#julia> Pkg.clone("https://github.com/hshindo/Merlin.jl.git")
```
